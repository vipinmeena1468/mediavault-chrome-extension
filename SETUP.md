# MediaVault – Setup Instructions

## 1. Download JSZip (required for ZIP feature)
Download `jszip.min.js` from: https://cdnjs.cloudflare.com/ajax/libs/jszip/3.10.1/jszip.min.js
Place it in the same folder as `manifest.json`.

## 2. Load the Extension in Chrome
1. Open Chrome → go to `chrome://extensions`
2. Enable **Developer Mode** (top right toggle)
3. Click **Load unpacked**
4. Select this folder (`media-extractor/`)

## 3. Usage
- Click the ⚡ MediaVault icon in the toolbar
- Click **Scan Page** on any webpage
- Switch tabs: Images / Videos / Audio / Background
- Click cards to select, then use **⬇ ZIP** to bulk download
- Use **⎘ URL** on any card to copy its URL to clipboard
